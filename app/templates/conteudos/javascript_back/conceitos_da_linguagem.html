Entendendo os conceitos

O que é JavaScript

JavaScript é uma linguagem de programação usada principalmente para criar interatividade e comportamento dinâmico em páginas e aplicações web. Ela permite reagir a eventos (cliques, teclas, rolagem), manipular elementos da página e consumir dados de APIs.

JavaScript também pode ser usado fora do navegador, especialmente com Node.js, para criar servidores, ferramentas de linha de comando e automações, mas nesta missão o foco é a base da linguagem e o que você precisa para programar com segurança.

Como o código funciona

Em JavaScript, o programa é executado de cima para baixo, instrução por instrução, salvo quando você cria desvios de fluxo (condições e laços) ou trabalha com funções e eventos.

Conceitos essenciais:

  • Instruções e blocos: instruções terminam com ; (opcional, mas recomendado em muitos contextos) e blocos usam

  • Comentários: // comentário e /* comentário em bloco */.

  • Case-sensitive: nome é diferente de Nome.

  • Console:console.log() é a forma mais comum de exibir valores durante o aprendizado.
Exemplo simples:
js // Uma mensagem no console console.log("Olá, JavaScript!");

Variáveis, tipos e operadores

Variáveis guardam valores para serem usados e modificados durante o programa. Declaração de variáveis



let: variável que pode mudar (recomendado no dia a dia). const: “constante” (o valor não pode ser reatribuído). var: forma antiga; evite em projetos modernos. Exemplo: js let idade = 16; const escola = "EREMCC"; idade = idade + 1; Tipos de dados principais number: números (inteiros e decimais). string: texto. boolean: true ou false. null: ausência intencional de valor. undefined: valor ainda não definido. object: estrutura (inclui objetos, arrays, datas etc). Observação importante: JavaScript tem tipagem dinâmica, então uma variável pode receber valores de tipos diferentes em momentos diferentes, mas isso exige atenção para evitar bugs. Operadores mais usados Aritméticos: + - * / % Comparação: == (evite), === (recomendado), != (evite), !== Lógicos: && (E), || (OU), ! (NÃO) Dica prática: prefira sempre === e !== para comparações, porque eles comparam valor e tipo, reduzindo “surpresas” do JavaScript. Controle de fluxo e funções Controle de fluxo define o caminho que o programa segue. Condicionais js const nota = 7; if (nota >= 7) { console.log("Aprovado"); } else { console.log("Reprovado"); } Laços de repetição for: quando você sabe quantas vezes repetir. while: repete enquanto uma condição for verdadeira. js for (let i = 1; i <= 3; i++) { console.log("Volta:", i); } Funções Funções organizam o código em partes reutilizáveis. js function somar(a, b) { return a + b; } console.log(somar(2, 3)); Também existe a arrow function: js const dobrar = (n) => n * 2; console.log(dobrar(5)); Objetos, arrays e o “mundo web” Objetos Objetos agrupam dados e comportamento. js const aluno = { nome: "Ana", turma: "2º Ano", aprovado: true }; console.log(aluno.nome); Arrays Arrays são listas ordenadas. js const linguagens = ["HTML", "CSS", "JavaScript"]; linguagens.push("Python"); console.log(linguagens.length); DOM (o que liga JS à página) No navegador, JavaScript conversa com a página via DOM (Document Object Model), que representa os elementos HTML como objetos manipuláveis. Exemplo (conceitual): Selecionar um elemento Alterar texto/estilo Reagir a um clique js const botao = document.querySelector("#meuBotao"); botao.addEventListener("click", () => { alert("Você clicou!"); }); Boas práticas iniciais Dê nomes claros para variáveis e funções (calcularMedia, totalCarrinho). Evite “mágica”: transforme números e textos importantes em constantes. Quebre o problema em funções pequenas. Use console.log() para entender valores antes de tentar “adivinhar” o erro.
