Entendendo os conceitos

O que é JavaScript

JavaScript é uma linguagem de programação usada principalmente para criar interatividade e comportamento dinâmico em páginas e aplicações web. Ela permite reagir a eventos (cliques, teclas, rolagem), manipular elementos da página e consumir dados de APIs.

JavaScript também pode ser usado fora do navegador, especialmente com Node.js, para criar servidores, ferramentas de linha de comando e automações, mas nesta missão o foco é a base da linguagem e o que você precisa para programar com segurança.

Como o código funciona

Em JavaScript, o programa é executado de cima para baixo, instrução por instrução, salvo quando você cria desvios de fluxo (condições e laços) ou trabalha com funções e eventos.

Variáveis, tipos e operadores

Variáveis guardam valores para serem usados e modificados durante o programa.

Declaração de variáveis:

  • let: variável que pode mudar (recomendado no dia a dia).
  • const: “constante” (o valor não pode ser alterado).
  • var: forma antiga; evite em projetos modernos.

Tipos de dados principais:

  • number: números (inteiros e decimais).
  • string: texto.
  • boolean: true ou false.
  • null: ausência intencional de valor.
  • undefined: valor ainda não definido.
  • object: estrutura (inclui objetos, arrays, datas etc).
Observação importante: JavaScript tem tipagem dinâmica, então uma variável pode receber valores de tipos diferentes em momentos diferentes, mas isso exige atenção para evitar bugs.

Operadores mais usados:

  • Aritméticos: +(soma), - (subtração), * (multiplicação), / (divisão), % (resto da divisão).
  • Comparação: == (igual), === (identico em valor e tipo), != (diferente), !== (diferente em valor e tipo).
  • Lógicos: && (E), || (OU), ! (NÃO).
Dica prática: prefira sempre === e !== para comparações, porque eles comparam valor e tipo, reduzindo “surpresas” do JavaScript.
